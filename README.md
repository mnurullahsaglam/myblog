## My Blog

This is me, Mehmet Nurullah Sağlam, a software engineer and a web developer. I write about web development, books,
software engineering, and the Laravel framework.

I have a WordPress blog, but as I am a web developer, I wanted to have my own blog. I have been using Laravel for a
while, so I decided to use it for my blog. I thought this shame to use a simple WordPress is enough. I wanted to have a
blog that I can customize and extend. I wanted to have a blog that I can use as a playground for my experiments. So here
we are. I'm planning to write my blog in three steps.

First, I will write a simple blog with the basic features. I will use the Laravel as the back-end framework and Tailwind
CSS as the frontend framework.

Second, I will write a blog with the basic features and some advanced features. I will use the Laravel as the back-end
framework and Vue.js and Tailwind as the frontend frameworks.

Finally, I hope to achieve building a personal blog with my "Golden 5" which are Laravel, Laravel Octane, Tailwind,
Inertia.js and Vue.js. Yeah, I know, it's a lot of frameworks, but I want to learn them all. I will write about them,
#builtinpublic.
